For Export Javascript Files 
In One File

steps :

1- first download and install "Node Js" -> https://nodejs.org/dist/v16.13.0/node-v16.13.0-x64.msi
2- run this command in your command line -> npm install --global gulp-cli
3- for more information about "Gulp Js" please go to this repository and read it -> https://gulpjs.com/docs/en/getting-started/quick-start

4- you should move your "js" files in -> src/js

5- you have three options here :

- run in your command line -> gulp 
or
- run -> gulp prod 
or 
- run -> gulp dev 

if you just run "gulp" command , gulp will create two file from your Javascript files
one development file 
two prodution file
